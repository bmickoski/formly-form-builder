# Security Policy

## Supported versions

Security fixes are applied to the latest `master` branch.

## Reporting a vulnerability

Please do not open public issues for security vulnerabilities.

1. Use GitHub Security Advisories for this repository.
2. If unavailable, contact the maintainer directly and include:
   - impact summary
   - reproduction details
   - potential fix suggestion

## Response targets

- Initial triage: within 3 business days
- Mitigation plan: within 7 business days for validated issues
- Public disclosure: coordinated after a fix is available
